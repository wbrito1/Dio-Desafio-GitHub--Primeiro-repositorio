Comandos básicos do Git

Windows
-cd = acessar uma pasta
-dir = listar pastas
-mkdir = criar um repositorio
-del/ rmdir = deletar coteudo de uma pasta/ deletar uma pasta

Unix
-cd = acessar uma pasta
-cd .. = volta um nivel
-ls = listar pastas
-ls -a = mostra arquivos ocultos
-mkdir = criar uma repositorio
-rm -rf = deletar uma pasta
-git init = iniciar repostiorio do Git dentro da pasta
-git add = mover arquivos, inicio ao versionamneto
-git add * = adicona arquivos
-git add. = adiciona todas as modificações pata Staged
-git commit = criar um commit
-git config --global user.email = criar email atrelado ao commit
-git config --global user.name =  criar autor atrelado ao commit
-git config --global --unset user.email/user.name = para deletar e alterar email e nick name
-git commit -m "" = criando um commit/ snapshot
-git status = monitorar o status dos arquivos
-mv arquivo ./repositorio = move arquivo ou repositorio para outro repositorio
